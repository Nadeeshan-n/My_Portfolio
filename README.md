#My_Portfolio

I developed my portfolio  using React and TypeScript. This project showcases projects, skills, and personal information with smooth animations and a clean UI.

---

##  Features

*  Built with React + TypeScript
*  Styled using Tailwind CSS
*  Component-based architecture
*  Smooth scroll & fade-in animations
*  Fully responsive design
*  Clean and maintainable code structure
*  Ready for deployment (Vercel / Netlify / GitHub Pages)

---

## Tech Stack

* **Frontend:** React, TypeScript
* **Styling:** Tailwind CSS
* **Icons:** Lucide React
* **Build Tool:** Vite

---

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
